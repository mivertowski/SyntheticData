# v1.1.0 Release Plan — Entity-Aware Anomalies, Stub Wiring & Quality

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Wire all dead-code stubs into active generation paths, deliver entity-aware anomaly injection as a coherent new feature, and close quality gaps (clippy pedantic, test coverage for 4 weak crates).

**Architecture:** Three pillars executed in dependency order: (1) Wire standalone generator stubs that have no cross-dependencies, (2) Wire the three anomaly injector stubs into the injection loop as a cohesive entity-aware anomaly feature, (3) Clippy cleanup and test expansion. Each task is independently committable.

**Tech Stack:** Rust 1.88+, ChaCha8Rng, rust_decimal, chrono, serde/serde_yaml

    for result in correlated {
        // Queue the correlated anomaly for injection on a future entry
        // (same entity if result.same_entity, after result.lag_days)

}
